Nerro Notes (Mobile Application)

Nerro Notes combines the functionality of a full text editor and voice recording allowing for an increase in productivity. Nerro Notes helps keep order in your lectures and meetings.


(Currently In Development) https://github.com/cmfnels/Nerro

Features:

- Rich editor support
- Sync text with audio for quick reference.
- Add and edit notes
- Add and edit timestamps


Highlights:

  • Android Studio
  • Java
  • Android SKD
  • SQL
  • Database
